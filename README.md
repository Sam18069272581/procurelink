# ProcureLink

Campus procurement price benchmarking tool for HSBI Bielefeld.  
Scrapes [Geizhals.de](https://geizhals.de) for B2B-relevant office supplies and visualises
cross-vendor price comparisons.

## Quick start

```bash
pip install requests beautifulsoup4 lxml
python crawler.py          # writes data.json
# then open index.html in your browser
```

## Deploy to GitHub Pages (free, auto-updates)

1. Push this repo to GitHub
2. Go to **Settings → Pages → Source: Deploy from branch (main, / root)**
3. GitHub Pages will serve `index.html` reading `data.json` from the same directory
4. The workflow in `.github/workflows/crawl.yml` runs every Monday at 08:00 Berlin time,
   crawls Geizhals.de, and commits the updated `data.json` back — fully automatic

## File structure

```
procurelink/
├── index.html              # Frontend (reads data.json, no server needed)
├── crawler.py              # Python scraper — outputs data.json
├── data.json               # Generated by crawler (committed to repo)
├── requirements.txt
└── .github/
    └── workflows/
        └── crawl.yml       # GitHub Actions schedule
```

## Customise targets

Edit the `TARGETS` list in `crawler.py`:

```python
TARGETS = [
    ("SKU-001", "Druckerpapier A4 80g (500 Bl.)", "Büromaterial", "druckerpapier a4 80g 500"),
    # add your own SKUs here
]
```

## data.json schema

```json
{
  "generated_at": "2025-04-19T06:00:00Z",
  "source": "geizhals.de",
  "product_count": 8,
  "results": [
    {
      "sku": "SKU-001",
      "name": "Druckerpapier A4 80g (500 Bl.)",
      "category": "Büromaterial",
      "offers": [
        { "shop": "Merchant GmbH", "price": 3.20, "url": "https://..." }
      ],
      "best_price": 3.20,
      "best_shop": "Merchant GmbH",
      "scraped_at": "2025-04-19T06:01:12Z",
      "error": null
    }
  ]
}
```
